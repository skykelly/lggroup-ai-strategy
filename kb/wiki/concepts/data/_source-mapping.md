# 업로드 파일과 Concept 매핑

| 파일 | Concept 도출에 사용한 정보 |
|---|---|
| `keywords.yml` | 검색 의도, 콘텐츠 제목, 관련 문서 |
| `search-keyword-matrix.json` | 검색어 클러스터, 상업 키워드, Concept alias 후보 |
| `spaces.yml` | 공간 역할, 관련 상품 카테고리, 관련 스타일 |
| `styles.yml` | 스타일 표준명, 키워드, 적용 공간, 관련 테마 |
| `tag-dictionary.json` | 표준 공간·스타일·테마·상품 그룹 태그 |
| `taxonomy.yml` | 문서 유형, 사용자, 상태 정의 |
| `themes-2026.yml` | 2026 테마 정의, 관련 스타일·공간, 비즈니스 시사점 |
