# Korea Interior Trend Wiki

한국 인테리어 스타일과 트렌드에 대한 마크다운 기반 위키입니다.

## 목적

브랜드·MD·마케터가 한국 리빙 시장의 스타일, 공간 트렌드, 카테고리 언어, 콘텐츠 키워드를 이해하고 활용할 수 있도록 구조화된 문서 체계를 제공합니다.

## 핵심 사용자

- 브랜드 담당자
- MD / 상품기획자
- 마케터 / 콘텐츠 기획자

## MVP 구성

- 연도별/시기별 트렌드 8개
- 2026 인테리어 테마 7개
- 스타일 사전 10개
- 공간별 트렌드 10개
- 카테고리/브랜드/마케팅 인사이트 5개
