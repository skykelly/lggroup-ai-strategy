## 2. Evidence Abstract

LG CNS와 Palantir는 전략적 파트너십을 체결했다. Foundry는 기업 내 분산 데이터를 통합·정제해 데이터 기반 운영체계를 구축한다. AIP는 통합 데이터 환경과 생성형 AI를 결합해 실시간 의사결정을 지원한다. LG CNS는 FDE 조직을 만들고 제조, 에너지, 전자, 물류 등 산업별 고부가 AX 프로젝트를 추진한다. LG그룹 내 적용을 시작점으로 외부 AX 사업 확장을 계획한다. Foundry는 분산 기업 데이터를 통합·정제해 데이터 기반 운영체계를 구축한다. LG그룹 내 적용을 시작으로 외부 AX 사업 확장을 계획한다.
